2017-2018学年度国资中心职工考核标准

考核评价项目 考核评价内容 评分
一级指标 二级指标
德
（15分） 		思想觉悟 积极参加中心组织的学习教育、培训活动，注重个人修养。顾全大局讲政治，践行国资中心核心价值观。（5分）
服务意识 树立为基层服务的观念，有较强的服务意识，基层单位反映良好。热情接待来人来访，言行举止规范得体，及时处理和解决问题。（5分）
团结协作 团结互助，互敬互谅，讲究合作。（5分）
能
（20分） 		工作能力 业务能力强，能够把握操作流程和关键环节。考虑问题细致周到，对突发事件有一定的判断能力和处理能力。（5分）
立足本岗，善于发现和思考新问题，提出意见和建议。（5分）
创新能力 主动学习本岗业务，参加业务培训，加强工作思考，不断提高业务水平和履职能力。（5分）
能够采取多种形式主动学习理论知识、管理知识和业务知识，虚心求教，不断提高履职能力，解决工作中遇到的实际问题。（5分）
勤
（15分） 		工作量 服从工作安排，承担岗位工作及组织交办的临时性工作。（5分）
工作态度 遵守工作纪律及《员工守则》，履行请假手续，遵守作息时间,不迟到、不早退、不串岗、不脱岗，不做与工作无关的事。对工作认真负责，积极主动，勤奋敬业。 （5分）
工作作风 实事求是，办事公正，坚持原则，处理问题及时有效。勇于批评和自我批评，敢于承担责任。（5分）
绩
（40分） 		工作质量 在规定时限内积极主动完成工作，做好相关沟通协调工作，效果良好，服务对象满意。（10分）
工作程序 严格履行规定工作程序及流程，特别在工程洽商、项目变更、租金减免、委托代办等事项杜绝随意性和个人行为。（10分）
工作效果 圆满完成年初指定的各项目标和任务，完成好各项临时性任务，完成“急难险重”任务情况，对科室、中心稳定与发展的贡献。（10分）
工作成绩 组织领导本部门工作取得的成绩和荣誉及本人受到的表彰（以证书和奖状为准）。（10分）
廉
（10分） 		党风廉政 认真执行党风廉政建设各项规定及工作要求，落实一岗双责。（5分）
廉洁从业 遵守各项法律法规，遵守并严格执行区教委及中心各项规章。廉洁奉公，能够严格要求自己，自觉维护本单位工作人员形象。（5分）
综合评价：德、能、勤、绩、廉（100分） 0
注：考核等次：合格及以上：75分（含）以上；基本合格：60—74分（含）；不合格：59分以下。

*注：请认真核对评分，一经提交不可修改！
